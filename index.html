# 🎯 轴承计算器 基于**GB/T 24607-2023**和**ISO 281**标准的圆锥滚子与角接触球轴承复合计算器。 ## ✨ 功能特性 - ✅ 角度换算工具（度分秒↔十进制） - ✅ 圆锥滚子轴承计算 - ✅ 角接触球轴承计算 - ✅ 载荷与寿命计算 - ✅ 移动端优化 - ✅ PWA支持（可添加到桌面） ## 🔧 使用技术 - HTML5 - CSS3 (Grid, Flexbox) - JavaScript (ES6+) - PWA (Progressive Web App) - LocalStorage (数据存储) ## 🚀 快速部署 1. 将此仓库Fork到您的GitHub账户 2. 在仓库Settings中开启GitHub Pages 3. 访问 https://您的用户名.github.io/仓库名 ## 📱 添加到手机桌面 - **iPhone**: Safari → 分享 → 添加到主屏幕 - **Android**: Chrome → 菜单 → 添加到主屏幕 ## 📄 相关标准 - GB/T 24607-2023 滚动轴承 寿命可靠性试验及评定方法 - ISO 281:2007 滚动轴承额定动载荷和额定寿命 - GB/T 6391-2010 滚动轴承 额定动载荷和额定寿命
